# Human-Activity-Recognition-using-Machine-Learning

Machine learning project to classify human activities using sensor-based time-series data with feature engineering and optimized models.

📌 Overview:
This project focuses on building a machine learning model to automatically classify human activities such as walking, sitting, and running using sensor-based time-series data from accelerometer and gyroscope signals. The system aims to enable intelligent activity tracking for real-world applications like fitness monitoring and wearable devices.

🚀 Features:
Data preprocessing and cleaning of sensor-based time-series data
Feature engineering using statistical and frequency-domain features
Implementation of multiple ML models:
   Logistic Regression
   K-Nearest Neighbors (KNN)
   Decision Trees
Model evaluation using accuracy, precision, recall, and F1-score
Performance improvement through normalization and hyperparameter tuning

⚙️ Tech Stack:
Language: Python
Libraries: NumPy, Pandas, Scikit-learn

📊 Workflow:
Data Collection (sensor dataset)
Data Preprocessing & Cleaning
Feature Engineering
Model Training
Model Evaluation
Performance Optimization

📈 Results:
Achieved reliable classification performance across multiple machine learning models. Feature engineering and normalization significantly improved the accuracy and generalization of the models.

💡 Applications:
Fitness tracking systems
Health monitoring solutions
Smart wearable devices
Activity-based recommendation systems
